## nc-widgets
### Usage
使用bower来安装：
```javascript
bower install --save git://github.com/HelloYu/nc-widgets.git
```
之后引用build文件夹内的nc-widgets.min.js，注入需要使用的模块：
```javascript
angular.module('app',['nc-treetable']);
```
其它内容参看目录docs。
